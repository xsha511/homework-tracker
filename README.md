# 寒假学习大礼包 📚

上海赫贤学校 2026 年春季 IGCSE 4.5年制新生寒假学习资源网站。

## 在线访问

🔗 **[xsha511.github.io/homework-tracker](https://xsha511.github.io/homework-tracker)**

## 功能特性

### 学习资料库
- 8 大学科完整学习资料（英语、中文、数学、物理、化学、生物、经济、地理）
- PDF 教材在线预览
- 详细作业说明和要求

### 任务清单系统
- **按日计划** - 22 天学习日程，每日任务一目了然
- **按学科分类** - 快速查看各科作业进度
- **进度追踪** - 实时显示完成百分比
- **本地存储** - 进度自动保存，刷新不丢失

### 激励系统
- **等级成长** - 从「学习新手」升级到「赫贤之星」
- **连续打卡** - 记录每日学习streak
- **成就徽章** - 6 类徽章，解锁学习里程碑

## 技术栈

- 纯静态网站（HTML + CSS + JavaScript）
- PDF.js 文档预览
- localStorage 数据持久化
- 响应式设计，支持手机浏览
- 阿里云 OSS 托管大型 PDF 文件

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/xsha511/homework-tracker.git

# 启动本地服务器
cd homework-tracker
python3 -m http.server 8000

# 访问 http://localhost:8000
```
